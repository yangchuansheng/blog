## 1. 回顾

不知不觉云原生实验室的同名博客网站 [https://icloudnative.io](https://icloudnative.io/) 已经上线一年了。今天有必要回顾一下这一年经历的事情。

+ **2019.10.20** : 第一次提交博客代码到 GitHub 并部署。
+ **2019.11.30** : 网站迁入[狗云](https://www.dogyun.com/?ref=ryanyang)。
+ **2019.12.19** : 网静态资源分离，增加 CDN 分发。
+ **2020.01.11** : 增加暗黑主题，优化阅读体验。
+ **2020.05.26** : 网站迁入[腾讯云](https://www.aliyun.com/minisite/goods?userCode=9j2ttg1g)。
+ **2020.07.19** : PV访问单日突破 `1800` 人次。
+ **2020.09.20** : 总访问次数突破 `150000` 次。

截止目前原创文章总数 150 篇，大部分都是云原生相关的文章，脚步还在一步一步向前。

## 2. 艰难的前进之路

这种成绩其实甚至算不上成绩，甚至没有什么亮点，个人网站已经没有什么可玩性了，但是为什么我还要坚持搞到今天？我觉得如果一个人能够把自己的爱好当作事业，那就是最大的幸福。我是云原生狂热信徒，同时又喜欢装逼，那么怎样才能在痴迷于云原生的同时又能装逼成功？于是我就想到了自媒体。这样一来，既能持续投身于爱好中，又能装逼，还特么能挣钱，坚持不下去才怪呢！

### 学习 SEO 推广

在网站刚上线的时候，访问量一个月内几乎是没有的，自己想办法去简单做了一下 SEO，同时也自己在圈子内推广了一波，逐渐有了访问量。这让我体会到很多互联网产品在推广初期太难了，如果没有好的运营思路和推广将会举步维艰。从中我也学会了简单的 SEO 和建站技巧。

### 学会分析运营数据

后面针对网站做了一些埋点来监控运营效果，自己没有运营的思路，就埋点这样就可以对流量进行溯源，包括访问的高峰期，流量入口，内容的热点，同时统计打开率和跳出率。然后针对性的开展一些运营策略。

### 从运营中提炼需求

跳出率过高，那我就增加搜索来帮助你精确查找内容，增加一些文档工具来保证打开率。到了后面甚至有人反映在某些地区访问不到，经常掉线。我又增加了 CDN  分发来保证响应速度。这一步一步有时候自己能想到，等真正的去做的时候发现还真的不容易。除了技术之外，还有很多因素影响，比如互联网法律法规方面，如何去备案，如何去获得赞助等等。

## 3. 最大的收获

作为云原生爱好者，同时也作为内容创作者，一个人支撑着这个小破站。从中学到了一整套的知识，实现了让自己的知识落地，这是最大的收获。

最开始也犹豫过自己能不能行，但是我觉得在成本可控的情况下作为一个 YAML 工程师一定要尝试一下，去验证一下自己的想法，无论是提高技术还是转行创业都有帮助。
